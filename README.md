# Grab-Mouse-Position.
This is a simple program in C++ Raylib that grabs the mouse Position.

This is my second project in my entire C++ Journey. WELL second in release I guess. Half of my projects was finding out Object orientated programming.
The program grabs the mouse position as a vector in 2d every click and sends it to the console for you to read.

I cannot say this will be useful for you but I'm a six grader who doesn't know any math which makes this very useful <3
